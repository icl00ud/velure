# Docker Compose - Local Development

Este arquivo `.env` é necessário para rodar o Docker Compose da raiz do projeto.

Os valores reais estão em `infrastructure/local/.env`

Para editar as variáveis de ambiente, use:
```bash
vim infrastructure/local/.env
```

Para sincronizar mudanças com a raiz:
```bash
cp infrastructure/local/.env .env
```

**Nota:** Este arquivo é automaticamente ignorado pelo Git (não será commitado).
